# Task FA-LOC-001: تنظیم فارسی به‌عنوان زبان پیش‌فرض و تکمیل ترجمه‌ها

## توضیح کوتاه
- تنظیم `fa` به‌عنوان زبان پیش‌فرض (fallback) در سیستم i18n
- تکمیل/بازبینی ترجمه‌های فارسی برای تمام صفحات اصلی

## Acceptance Criteria
- بدون نیاز به انتخاب زبان، UI پیش‌فرض فارسی باشد.
- برای تمام کلیدهای i18n استفاده‌شده در UI، مقدار فارسی موجود باشد یا حداقل متن انگلیسی fallback نمانده باشد.

## گام‌های پیشنهادی
- بررسی پکیج `ra-language-farsi` و مپ‌کردن آن با ساختار فعلی ترجمه‌ها.
- اضافه/تکمیل فایل ترجمه `fa` در `src/i18n`.
- تنظیم `defaultLocale` / `fallbackLng` روی `fa`.
- غیرفعال یا override کردن auto-detect زبان مرورگر (در صورت وجود).
