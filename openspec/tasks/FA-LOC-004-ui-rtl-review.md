# Task FA-LOC-004: پشتیبانی RTL و بازبینی ظاهری UI

## توضیح کوتاه
- اعمال `dir="rtl"` و استایل‌های لازم برای پشتیبانی کامل راست‌به‌چپ.
- بازبینی صفحات کلیدی برای اطمینان از صحت نمایش.

## Acceptance Criteria
- بدنه اپلیکیشن در حالت عادی RTL باشد و متن‌ها راست‌چین نمایش داده شوند.
- آیکون‌ها و ترتیب اجزا در کامپوننت‌های مهم (مثلاً دکمه‌های back/next، جدول‌ها، فرم‌ها) در RTL منطقی و قابل‌استفاده باشند.

## گام‌های پیشنهادی
- تنظیم `dir="rtl"` روی روت اپ (مثلاً `<html>` یا `<body>` یا روت ری‌اکت).
- تعریف کلاس/تم RTL در استایل‌ها و اعمال روی layout اصلی.
- بازبینی صفحات: Login، داشبورد، مدیریت کاربران، مدیریت روم‌ها.
- رفع مشکلات ریز (overlap متن و آیکون، alignment جدول‌ها، breadcrumbها، ...).
