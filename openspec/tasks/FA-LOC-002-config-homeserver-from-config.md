# Task FA-LOC-002: تنظیم homeserver از کانفیگ و حذف ورودی آن

## توضیح کوتاه
- خواندن آدرس homeserver از فایل کانفیگ یا env
- حذف فیلد ورودی homeserver از صفحه ورود/اصلی و استفاده از مقدار کانفیگ در کلاینت

## Acceptance Criteria
- هیچ ورودی قابل ویرایشی برای آدرس homeserver در UI وجود نداشته باشد.
- همه‌ی درخواست‌ها به Synapse از آدرسی استفاده کنند که در کانفیگ مشخص شده است.

## گام‌های پیشنهادی
- تعریف کلید جدید در کانفیگ (مثلاً `homeserverUrl` یا متغیر env).
- ایجاد/به‌روزرسانی ماژول Config برای اکسپوز کردن این مقدار.
- استفاده از این مقدار در لاجیک ساخت client و توابع `getServerVersion` و ...
- تمیز کردن هرگونه state/prop مربوط به `base_url` که از کاربر گرفته می‌شد.
